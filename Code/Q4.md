### Idea 3
    
    Sliding widonw
- Para cada i buscar el primer j talque cada uno de sus elementos tenga por lo menos m y que exitan k elementos distintos.
- Si para un i el primer indice que cumple es j entonces, para un i+1 el primer indice no podra ser menor que j. 
- Si para un i el primer indice no existe j entre el rango donde hat k distintos, entonces para i+1 no existira un indice menor que j talque cumpla la condicion.
